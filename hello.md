This project is called zClone. We are making a chatgpt clone that is based upon https://github.com/zed-industries/zed/tree/main/crates/gpui

This makes sures it is a smooth and fast experience. 

It has all the features as you would expect from a chatgpt clone, with chat windows, with side panel (collapsable) for all the chats. In the setting section give a place for the opeai api key, the user can bring to make this app work.

Keep the UI inspiring, use modern fonts that will look good on MAC, and make it sleek, fast and smooth. Have dark and light mode.
